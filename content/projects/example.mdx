This is an example project file. You can use this as a template to create your own projects.

## How to create a new project

1. Create a new `.mdx` file in the `content/projects/` directory
2. Add the frontmatter with all the required fields
3. Write your content in Markdown below

## Features

- **Markdown support**: Write your content using Markdown
- **Code blocks**: Use code blocks with syntax highlighting
- **Links**: Add links to your project website or repository

```typescript
// Example code block
const example = "This is TypeScript code";
console.log(example);
```

## Metadata fields

- `title`: Project title (required)
- `description`: Short description (required)
- `date`: Publication date (optional)
- `published`: Set to `true` to show the project (optional)
- `url`: Link to the project website (optional)
- `repository`: GitHub repository in format `owner/repo` (optional)
